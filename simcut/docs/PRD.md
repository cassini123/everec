## 1. 一句话定位

**中文：** Simcut — 对标剪映的**轻量剪辑软件**，但更稳定、渲染更快，专为**超短篇**创作设计。

**英文：** Simcut — A lightweight video editor for ultra-short content. Faster, more stable than CapCut.

**核心价值：**

| 维度 | Simcut | 剪映 |
|------|--------|------|
| 稳定性 | Rust 核心 + FFmpeg 管线，导出成功率优先 | 功能堆叠，偶发崩溃 |
| 渲染速度 | 代理预览 + fast preset，超短篇快出片 | 全功能渲染较重 |
| 文件管理 | 项目制组织素材/静帧/LUT/字幕 | 素材库分散 |
| 色彩设计 | 上传参考图 → 自适应 LUT | 滤镜预设为主 |
| 结项速度 | 一键导出相册 + 网盘 | 多步骤分享 |

---

## 2. 目标用户

- 短视频 / 超短篇创作者（30s–30min）
- 需要快速出片、稳定交付的个体创作者
- 对色彩有审美要求但不想深入调色软件的用户

---

## 3. 功能清单（v0.1）

### 3.1 多语言自动字幕识别

- 支持中文、英文、日文、韩文
- 自动语音识别 → 字幕轨道
- 字幕与时间轴联动

### 3.2 色彩解析

- 上传参考照片
- 识别主色调与色彩系统（色温、对比度、饱和度）
- 生成自适应 LUT（.cube）并一键应用

### 3.3 静帧管理

- 在时间轴任意位置捕捉静帧
- 关联色彩参考与风格标签
- 静帧库按项目组织

### 3.4 简单特效预设

- 渐显 / 渐隐
- 高亮强调
- 推近 / 背景虚化
- 一键应用到选中片段

### 3.5 自定义字体设计界面

- 字体族、字号、字重、字间距
- 颜色、描边、阴影
- 实时预览

### 3.6 AI 解析

- 屏幕显示 ISO、快门、光圈、曝光 EV、白平衡
- 高光溢出 / 阴影细节分析
- 调整建议
- Prompt 剪辑调整（自然语言 → 可执行编辑动作）

### 3.7 导出渲染

- 格式：MP4 / MOV / WebM
- 分辨率：720p / 1080p / 4K / 竖屏 9:16
- **导出直接保存相册**
- **勾选同时上传网盘**（iCloud / Google Drive / 百度网盘 / 阿里云盘）

## 4. 技术架构

```
simcut/
├── crates/
│   ├── timeline-engine/    # 项目图谱、时间轴、特效
│   └── color-engine/       # 色彩分析、LUT 生成
└── apps/
    └── desktop/            # Tauri 2 + React 19
        ├── src/            # Rust 后端命令
        └── ui/             # 共享 UI（桌面 + 网页）
```

| 层 | 技术 |
|----|------|
| 桌面壳 | Tauri 2 |
| 前端 | React 19 + TypeScript + Vite 6 + Tailwind 4 |
| 视频核心 | FFmpeg（渲染/export） |
| 数据 | 本地 JSON 项目文件，local-first |
| 网页端 | 同一 `ui/` 构建为静态 SPA |

### 桌面 vs 网页

- **桌面端（App）**：完整功能 — 媒体导入、渲染、相册保存、网盘上传
- **网页端**：UI 预览 + 部分功能降级（色彩分析示例、AI 分析 mock）

## 5. 非目标（v0.1）

- 完整 NLE 多轨专业剪辑
- 自研 GPU 合成器
- 复杂关键帧曲线
- 手机端原生剪辑（Companion 模式暂缓）

---

## 6. 稳定性策略（相对剪映）

1. **Rust 核心**处理项目状态与文件 IO，减少前端崩溃面
2. **FFmpeg 统一渲染管线**，避免多编码器路径
3. **代理预览**：低分辨率预览，导出时全质量
4. **导出任务队列**：失败可重试，状态可追溯
5. **本地优先**：不依赖云端即可完成闭环
